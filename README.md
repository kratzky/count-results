# Just an example of getting the total results number from google search

## Install dependencies

```sh
npm i
```

## Run

```sh
npm start
```

When done prints the result to console like:

```
┌─────────┬──────────────────┬─────────┐
│ (index) │       site       │ results │
├─────────┼──────────────────┼─────────┤
│    0    │  'example.com'   │  11240  │
│    1    │  'foobar'.com'   │  2323   │
└─────────┴──────────────────┴─────────┘
```
